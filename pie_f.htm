Filtering¶

With so much data in the file, we can filter it by selecting only the parameters of interest to us. We will select only individuals ages 0-17 and injuries resulting in deaths. We can then combine the pandas functions .groupby and .sum to get cumulative sums of the injury mechanisms that lead to death.

In [6]:
df = df.dropna()
df = df[df['age_group'] == '0-17']
df = df[df['type'] == 'Deaths']

df_young_deaths = df.groupby(['injury_mechanism']).sum()
