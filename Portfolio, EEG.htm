Background¶

This sample uses EEG data from an experiment looking to identify the brain's response to semantic violations across multiple conditions. For the purpose of this sample, we will focus only on visualizing the control condition. Using the Python package MNE, we can use topographic maps to look at the response following the stimulus.

Plotting the maps¶

Using a loop and the MNE package, we can plot through each condition of interest while specifying the time range of interest. We expect a much more negative response during the 400-600ms range (known as the N400 paradigm), so we will plot each 100ms for 300-700ms to visualize the difference.

In [32]:
time=np.arange(.300, .700+.001, .100)
for index in gavg.keys():
    gavg[index].plot_topomap(time, average=.200, title=index, vmin=-3, vmax =3)

Discussion¶

As seen in the topographic scalp maps above, the brain is considerably more negative in the violation condition, and peaks during the 400-600ms window indicating an N400 response may be present.
